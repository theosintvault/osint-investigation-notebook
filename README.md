# Investigation Notebook

A local-first, browser-native logbook for real-time investigative tracking and evidence persistence.

[🚀 Launch Notebook](https://theosintvault.io)

## Core Capabilities
* **Relational Logging:** Maintain a chronological sequence of subject pivots and lead developments.
* **Evidence Persistence:** Integrated local storage for source URLs and cryptographic hashes.
* **OpSec-Centric:** Zero-transmission architecture ensures all investigative data remains within the local environment.

## Technical Specifications
* **Environment:** Chromium-based browser (Recommended)
* **Storage Schema:** IndexedDB / LocalStorage
* **Connectivity:** 100% Offline-capable
